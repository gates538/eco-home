# Eco Home v1.1.2

> Versione archiviata. Per nuove installazioni usa la versione corrente nella cartella principale del repository.

## File

- `eco-home-v1.1.2.yaml`: automazione standalone della versione 1.1.2;
- `eco-home-v1.1.2-dashboard-card.yaml`: card della versione 1.1.2.

## Caratteristiche principali

- combinazione presenza-portone in entrambi gli ordini;
- finestra predefinita di 5 minuti;
- nomignoli casuali maschili, femminili e di coppia;
- gestione Nest Hub, TV, luce e asciugatrice;
- test vocale semplice.

La versione 1.1.3 aggiunge test per scenario, diagnostica persistente, ripristino del volume del Nest Hub e protezioni aggiuntive.
